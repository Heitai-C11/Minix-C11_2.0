whoami(1)

NAME

     whoami - print current user name

SYNOPSIS

     whoami

EXAMPLES


     whoami              # Print user name

DESCRIPTION


     In case you forget who you are logged in as, whoami will  tell  you.   If
     you  use  su  to  become  somebody  else,  whoami  will  give the current
     effective user.

SEE ALSO

     id(1), who(1).
