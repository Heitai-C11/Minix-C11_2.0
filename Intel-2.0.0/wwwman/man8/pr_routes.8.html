pr_routes(8)

NAME

     pr_routes - show IP routing.

SYNOPSIS

     pr_routes [-i ip device]

DESCRIPTION

     Pr_routes displays the IP routing table.

OPTIONS


     -i   ip device specifies the ip device.

SEE ALSO

     add_routes(8), irdp(8).
